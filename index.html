This project provides roughly 240gb of synthesiser samples, construction kits, loops, drum kits and other associated items. When appropriate, sampler patch files will be provided. The primary supported formats are .exs for Logic Pro X & .nki for Kontakt 5+.

Currently the site is undergoing maitainance. For a complete selection of sample libraries please visit our GitHub Project page GitHub Project page. To see an archived copy of this website with demo's and other features GitHub click here.
